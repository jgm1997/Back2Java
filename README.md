# Proyecto de Retorno a Java

Este proyecto está diseñado para ayudar a desarrolladores a retomar la programación en Java, utilizando Gradle como sistema de construcción y soporte para Kotlin.

## Características

- Código fuente en **Java**
- Gestión de dependencias y construcción con **Gradle**
- Estructura de proyecto moderna y lista para pruebas

## Requisitos

- JDK 21 o superior
- Gradle 9.x o superior

## Estructura del Proyecto

- `src/main/java` — Código fuente Java
- `src/test` — Pruebas unitarias

## Cómo compilar y ejecutar

```sh
gradle build
gradle run
